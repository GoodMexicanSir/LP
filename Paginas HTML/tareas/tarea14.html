Tarea 14

Problema PS3.2

Escribe un diagrama de flujo y el correspondiente programa en C que, al recibir como dato un número entero N, calcule el resultado de la siguiente serie:
1 + 1/2 + 1/3 + 1/4 + ... + 1/N

JAVA

import javax.swing.JOptionPane;
public class tarea14{
	public static void main (String[] args){
		int i, n;
		double a;
		String entrada;
		String serie = "";

		entrada = JOptionPane.showInputDialog("Ingrese el numero de terminos: ");
		n = Integer.parseInt(entrada);

		for(i=1; i< n; i++){
			a = 1.0/i;
			serie = serie + a + ", ";
		}
		a = 1.0/n;
		serie = serie + a + ".";
		JOptionPane.showMessageDialog(null,serie);
	} //Fin de función main
} //Fin de clase tarea14
